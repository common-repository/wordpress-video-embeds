WordPress Video Embeds (WP Video Embeds) is a plugin for WordPress to embed videos from many video sharing websites using shortcode.

== Description ==

WordPress Video Embeds (WP Video Embeds) is a plugin for WordPress to embed videos from many video sharing websites and a very useful plugin for video blogging as well as general blogging with videos..

This plugin uses shortcode to embed the video directly into a blog posts, Comments, RSS and even in custom excerpts.

== Installation ==

1. Upload "wp-video-embeds" to the "/wp-content/plugins/" directory.
1. Activate the plugin "WordPress Video Embeds" through the 'Plugins' menu in WordPress.
1. Adjust the Video Size from the "WP Video Embeds" option under Settings Menu.
